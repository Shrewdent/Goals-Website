GoalPost.io FAQ




Questions About The Site

What is a widget? A widget is a component of an interface which enables users to perform a function and/or action or access a service used. In GoalPost.io, a widget is a draggable element that a user can place in a blank workspace to add the goals needed to accomplish. They can always opt out of a goal or change it whenever it's not needed or if there is something else to do.

Which operating system can GoalPost.io run on? GoalPost.io can run on Microsoft Windows, Apple Mac OS X, Ubuntu Linux, Samsung Android, iOS, and any other OS that provides the features for the website.
